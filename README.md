# ruby-vm

## configure ssh agent forwarding
[Set up ssh key on host machine](https://help.github.com/articles/generating-a-new-ssh-key-and-adding-it-to-the-ssh-agent/)

Make sure ssh identity is available on host machine:

`ssh-add -l`
